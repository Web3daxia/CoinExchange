# 币本位合约模块API文档

## 基础信息

- **模块名称**：币本位合约模块（Coin-margined Futures）
- **用户端基础路径**：`/api/futures-coin`
- **后台管理基础路径**：`/admin/futures-coin`
- **认证方式**：Bearer Token (JWT)

**重要说明**：币本位合约模块已独立实现，不再共用U本位合约代码。保证金和盈亏以合约的基础币种结算（如BTC/USD合约以BTC结算），而非USDT。

---

## 核心交易规则

### 手续费计算

**公式**：
```
手续费 = 手续费率 × (成交张数 × 合约乘数 × 合约面值 × 成交价格)
```

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001，成交价格50000 USD
- 成交张数：10张
- Taker手续费率：0.05%
- 手续费 = 0.0005 × (10 × 0.0001 × 100 × 50000) = 0.25 USD

---

### 保证金计算

#### 全仓模式

账户里所有的可用资金都视作可用保证金。

**币本位合约 - 全仓初始保证金**：
```
初始保证金 = 面值 × |张数| × 合约乘数 / (标记价格 × 杠杆倍数)
```

**特点**：初始保证金将随交易币种价格变化而变动。

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001
- 开仓张数：10张
- 标记价格：50000 USD
- 杠杆倍数：10倍
- 初始保证金 = 100 × 10 × 0.0001 / (50000 × 10) = 0.002 BTC

#### 逐仓模式

各个仓位单独核算保证金，盈亏互不影响。

**币本位合约 - 逐仓初始保证金**：
```
初始保证金 = 面值 × |张数| × 合约乘数 / (开仓均价 × 杠杆倍数)
```

**特点**：初始保证金固定不变。

```
保证金余额 = 开仓保证金 + 手动追加（或减少）的保证金
```

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001
- 开仓张数：10张
- 开仓均价：50000 USD
- 杠杆倍数：10倍
- 初始保证金 = 100 × 10 × 0.0001 / (50000 × 10) = 0.002 BTC（固定）

---

### 维持保证金计算

**币本位保证金合约**：
```
维持保证金 = 面值 × |张数| × 合约乘数 × 维持保证金率 / 标记价格
```

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001
- 持仓张数：10张
- 维持保证金率：0.5%
- 标记价格：50000 USD
- 维持保证金 = 100 × 10 × 0.0001 × 0.005 / 50000 = 0.000001 BTC

**触发强平条件**：当保证金率低于维持保证金率+平仓手续费率时，即触发强平或强制部分减仓。

---

### 保证金率计算

#### 全仓模式

**现货和合约保证金 - 全仓**：
```
保证金率 = (该币种全仓余额 + 全仓收益 - 该币种挂单卖出数量 - 期权买单所需要的该币种数量 - 逐仓开仓所需要的该币种数量 - 所有挂单手续费) / (维持保证金 + 爆仓手续费)
```

#### 逐仓模式

**币本位保证金合约 - 逐仓**：
```
保证金率 = (保证金余额 + 收益) / (面值 × |张数| / 标记价格 × (维持保证金率 + 手续费率))
```

**示例**：
- 保证金余额：0.002 BTC
- 收益：0.0001 BTC
- 面值：100 USD，张数：10张，标记价格：50000 USD
- 维持保证金率：0.5%，手续费率：0.05%
- 保证金率 = (0.002 + 0.0001) / (100 × 10 / 50000 × (0.005 + 0.0005)) = 0.0021 / 0.0011 = 1.909（190.9%）

---

### 收益计算

#### 多仓收益

**币本位保证金合约**：
```
多仓收益 = 面值 × |张数| × 合约乘数 × (1/开仓均价 - 1/标记价格)
```

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001
- 持仓张数：10张
- 开仓均价：50000 USD
- 标记价格：51000 USD
- 多仓收益 = 100 × 10 × 0.0001 × (1/50000 - 1/51000) = 0.000039215 BTC

#### 空仓收益

**币本位保证金合约**：
```
空仓收益 = 面值 × |张数| × 合约乘数 × (1/标记价格 - 1/开仓均价)
```

**示例**：
- 合约：BTC/USD，面值100 USD，合约乘数0.0001
- 持仓张数：10张
- 开仓均价：50000 USD
- 标记价格：49000 USD
- 空仓收益 = 100 × 10 × 0.0001 × (1/49000 - 1/50000) = 0.000040816 BTC

#### 收益率

```
收益率 = 收益 / 开仓保证金
```

---

### 预估强平价计算

**单币种/跨币种/投资组合保证金模式 - 逐仓**：

#### 多仓预估强平价

**币本位保证金合约**：
```
多仓预估强平价 = 面值 × |张数| × (维持保证金率 + 手续费率 + 1) / (保证金余额 + 面值 × |张数| / 开仓均价)
```

**示例**：
- 面值：100 USD，张数：10张
- 维持保证金率：0.5%，手续费率：0.05%
- 保证金余额：0.002 BTC
- 开仓均价：50000 USD
- 多仓预估强平价 = 100 × 10 × (0.005 + 0.0005 + 1) / (0.002 + 100 × 10 / 50000) = 1005 / 0.02 = 50250 USD

#### 空仓预估强平价

**币本位保证金合约**：
```
空仓预估强平价 = 面值 × |张数| × (维持保证金率 + 手续费率 - 1) / (保证金余额 - 面值 × |张数| / 开仓均价)
```

**示例**：
- 面值：100 USD，张数：10张
- 维持保证金率：0.5%，手续费率：0.05%
- 保证金余额：0.002 BTC
- 开仓均价：50000 USD
- 空仓预估强平价 = 100 × 10 × (0.005 + 0.0005 - 1) / (0.002 - 100 × 10 / 50000) = -995 / (-0.018) = 55277.78 USD

### 资金费用

- **收取频率**：每8小时收取一次
- **收取时间**：8:00、16:00、24:00（UTC时间）
- **记录表**：`funding_fee_records`
- **计算方式**：根据资金费率、持仓张数、标记价格计算

**资金费用计算公式**（币本位合约）：
```
资金费用 = 面值 × |张数| × 合约乘数 × 资金费率 / 标记价格
```

### 2. 订单管理

#### 2.1 提交订单

**接口地址**：`POST /api/futures-coin/order`

**请求头**：
```
X-User-Id: 1
Content-Type: application/json
```

**请求参数**：
```json
{
  "pairName": "BTC/USD",
  "orderType": "LIMIT",
  "side": "BUY",
  "positionSide": "OPEN_LONG",
  "price": "50000.00000000",
  "quantity": "10.00000000",
  "leverage": 10,
  "marginMode": "ISOLATED"
}
```

**参数说明**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pairName | String | 是 | 交易对名称 |
| orderType | String | 是 | 订单类型：MARKET, LIMIT, STOP_LOSS, TAKE_PROFIT |
| side | String | 是 | 买卖方向：BUY, SELL |
| positionSide | String | 是 | 仓位方向：OPEN_LONG, OPEN_SHORT, CLOSE_LONG, CLOSE_SHORT |
| price | String | 条件 | 委托价格（限价单必填） |
| quantity | String | 是 | 委托数量（张数） |
| leverage | Integer | 是 | 杠杆倍数 |
| marginMode | String | 是 | 保证金模式：CROSS（全仓）, ISOLATED（逐仓） |

- **ISOLATED（逐仓）**：各个仓位单独核算保证金，盈亏互不影响
