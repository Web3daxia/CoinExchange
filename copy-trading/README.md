# 跟单交易模块 (Copy Trading Module)

## 模块概述

跟单交易模块允许平台用户根据策略选择跟随其他交易员（带单员）的操作。带单员将其交易策略进行公开或私密分享，跟单员可选择跟随，并在自己的账户中自动复制带单员的交易。系统支持现货和合约市场的跟单，并且可以区分公域跟单和私域跟单。

## 主要功能

### 1. 带单员管理
- **申请成为带单员**：用户提交申请，包括联系信息、资产证明等
- **审核机制**：后台审核带单员申请，设置资产门槛和保证金要求
- **保证金冻结**：成为带单员需要冻结保证金，退出时退还
- **带单员类型**：区分现货交易员、合约交易员或两者都是

### 2. 现货跟单
- 支持公域跟单和私域跟单
- 自动同步带单员的交易操作（开仓、平仓、止损、止盈）
- 支持资金分配和跟单比例设置
- 支持止损、止盈等风险控制参数

### 3. 合约跟单
- 支持USDT本位合约和币本位合约跟单
- 支持杠杆设置（全仓或逐仓）
- 支持杠杆倍数设置
- 支持止损、止盈、强平等风险控制

### 4. 公域跟单
- 策略公开，任何用户都可以选择跟随
- 可以设置交易佣金或平台分成
- 无门槛或低门槛参与

### 5. 私域跟单
- 通过邀请码或邀请制分享策略
- 可以设置订阅费或成功交易佣金
- 带单员可以指定跟单员、设置费用、限制跟单规模

### 6. 带单员等级与分润
- **等级划分**：初级、中级、高级、顶级
- **等级提升**：根据收益率、跟单人数、AUM、交易频率等因素动态调整
- **分润比例**：等级越高，分润比例越高（50%-80%）
- **收益分成**：带单员从跟单员的盈利中抽取一定比例的分润

### 7. 带单员表现数据
- **收益率**：带单员的整体收益率
- **总盈亏**：带单员的累计盈亏
- **胜率**：盈利交易占总交易的比例
- **总资产（AUM）**：带单员的资产管理规模
- **跟单人数增长**：展示跟单人数变化趋势
- **夏普比率**：衡量风险调整收益
- **最大回撤**：反映交易风险的指标
- **盈亏比**：每笔盈利与亏损的比例
- **交易统计**：总交易笔数、盈利笔数、亏损笔数、日均交易频次等

## API接口

### 带单员管理
- `POST /copy-trading/trader/apply` - 申请成为带单员
- `GET /copy-trading/market` - 查询交易员列表
- `GET /copy-trading/trader/{traderId}` - 查询带单员详情
- `POST /copy-trading/trader/invite` - 生成邀请码

### 跟单操作
- `POST /copy-trading/follow` - 跟随带单员
- `POST /copy-trading/stop` - 停止跟单
- `POST /copy-trading/pause` - 暂停跟单
- `POST /copy-trading/resume` - 恢复跟单
- `GET /copy-trading/account/status` - 查询跟单状态
- `POST /copy-trading/settings` - 设置跟单参数

### 带单员表现
- `GET /copy-trading/leaderboard` - 查询带单员排行榜
- `GET /copy-trading/leaderboard/{traderId}` - 查询单个带单员详细表现
- `GET /copy-trading/leaderboard/{traderId}/performance` - 获取带单员历史业绩数据
- `GET /copy-trading/leaderboard/performance/period` - 按周期查询带单员表现

## 带单员等级规则

### 等级划分
- **BEGINNER（初级）**：分润比例50%
- **INTERMEDIATE（中级）**：分润比例60%
- **ADVANCED（高级）**：分润比例70%
- **TOP（顶级）**：分润比例80%

### 等级提升条件
- AUM（资产管理规模）
- 收益率
- 胜率
- 跟单人数
- 交易频率

## 跟单流程

1. **带单员申请**：用户提交申请，包括联系信息、资产证明
2. **后台审核**：管理员审核申请，检查资产和资料
3. **冻结保证金**：审核通过后冻结保证金
4. **开启跟单**：带单员设置公域/私域跟单
5. **跟单员跟随**：跟单员选择跟随，设置资金分配和风险参数
6. **自动复制**：带单员下单时，系统自动复制到跟单员账户
7. **分润结算**：根据盈利情况计算并分配分润

## 风险控制

- **止损设置**：跟单员可以设置止损价格或百分比
- **止盈设置**：跟单员可以设置止盈价格或百分比
- **资金分配**：跟单员可以控制每个跟单策略的资金分配
- **跟单比例**：可以设置跟单比例（如50%复制）
- **暂停/停止**：跟单员可以随时暂停或停止跟单

## 注意事项

1. **带单员申请**：需要满足资产门槛，提交完整的联系信息和资产证明
2. **保证金管理**：成为带单员需要冻结保证金，退出时退还
3. **跟单风险**：跟单存在风险，跟单员应设置合理的止损和止盈
4. **分润机制**：分润比例根据带单员等级动态调整
5. **表现数据**：定期更新带单员的表现数据，帮助跟单员选择合适的带单员
